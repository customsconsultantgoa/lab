# Customs Consultant Goa

Sito vetrina statico realizzato con HTML, CSS e JavaScript vanilla.

## Anteprima locale

Aprire `index.html` con un browser.

## Configurazione email

Quando sara disponibile l'indirizzo definitivo, modificare questa riga in `script.js`:

```js
const CONTACT_EMAIL = "indirizzo@example.com";
```

Il form aprira l'app email dell'utente con oggetto e messaggio precompilati. Il sito non memorizza ne trasmette direttamente i dati.

## Pubblicazione su GitHub Pages

1. Creare un repository GitHub.
2. Caricare nella radice `index.html`, `style.css`, `script.js` e `.nojekyll`.
3. Aprire `Settings > Pages` nel repository.
4. In `Build and deployment`, scegliere `Deploy from a branch`.
5. Selezionare il branch `main`, cartella `/ (root)`, quindi salvare.

GitHub mostrera l'indirizzo pubblico dopo il completamento della pubblicazione.
